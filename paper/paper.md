# Summary

This is a minimal working example used to test PDF rendering for the Journal of Open Source Software (JOSS). The purpose of this example is to verify that `paper.md` and `paper.bib` are correctly formatted and that the JOSS rendering pipeline can generate a valid PDF.

This example cites one reference for testing citation rendering [@smith2020demo].

# Statement of Need

Researchers preparing a JOSS submission often need to confirm that their Markdown document renders correctly as a PDF. This minimal template provides a clean starting point for verifying the JOSS toolchain, GitHub Actions setup, and bibliography integration.
